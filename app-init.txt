App initialization
===================
1) Node init
npm init

2) Git init
echo "# poc-home-broker-vue-node" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/LeonardoNevesDuarte/poc-home-broker-vue-node.git
git push -u origin main

3) Dependencias do App
npm install websocket --save
npm install filehandler --save

4) frameworks install
- Bootstrap V.5.1.3 (download)
- Vue (download)
